# WordPress.org Asset Guide

This plugin repository contains runtime assets in `assets/css/`, but WordPress.org directory branding assets are uploaded separately in the plugin SVN `assets/` directory.

## Recommended Asset Filenames

- `banner-772x250.png`
- `banner-1544x500.png`
- `icon-128x128.png`
- `icon-256x256.png`

## Visual Direction

- Keep the background clean and commerce-oriented: navy, blue, and white already match the admin UI.
- Use a shopping cart or cart-with-clock symbol to communicate inactivity cleanup.
- Keep the wordmark short: `CartFlush`.
- Avoid too much small text because WordPress.org asset thumbnails shrink heavily.

## Suggested Banner Layout

- Left side: Cart icon with a subtle timer or refresh motif
- Right side: `CartFlush` wordmark plus one short supporting line
- Supporting line idea: `Auto-clear inactive WooCommerce carts`

## Icon Concept

- Rounded square background
- Cart outline in white
- Small timer or sweep accent in a lighter blue

- Use a clean WordPress admin environment with WooCommerce active.
- Hide unrelated admin notices before capture.
- Use realistic but non-sensitive sample data.
- Export PNG files sized clearly for WordPress.org listing previews.
