Automatically clears WooCommerce carts after a defined period of user inactivity to keep your store clean and optimized.

== Description ==

**CartFlush** helps WooCommerce store owners automatically empty abandoned carts after a configurable amount of time. This plugin ensures your store remains clean of old, abandoned sessions, enhancing performance and accuracy in reporting.

**Features:**

- Automatically clears WooCommerce cart after inactivity  
- Admin can define inactivity time (in minutes)  
- Lightweight and dynamic – no hardcoded values  
- Improves store performance and user session accuracy  

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/cartflush` directory, or install the plugin through the WordPress plugins screen directly.  
2. Activate the plugin through the 'Plugins' screen in WordPress  
3. Navigate to `WooCommerce > CartFlush Settings` to configure timeout options  

== Frequently Asked Questions ==

= Can I change the inactivity timeout? =  
Yes! You can set the inactivity timeout in minutes from the CartFlush settings page in the WordPress dashboard.

= Will this work for both logged-in and guest users? =  
Yes. The plugin detects inactivity based on session and user activity for both types.
